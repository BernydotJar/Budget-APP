# **App Name**: BudgetFlow

## Core Features:

- Transaction Entry: Allow manual entry of income and expenses with fields for date, amount, type, category, and account.
- Transaction Log: Display a list of all transactions with filtering options. Include editing and deleting functionalities.
- Dashboard Overview: Display key financial metrics (balance, income, expenses) and a pie chart of expenses by category.
- Reporting: Visual Summaries: Dedicated section with Pie Charts showing expense breakdown by category (allow filtering by date range and account)
- Category Management: CRUD operations for user-defined categories (e.g., Food, Transport, Salary) Connected with secure database
- Secure Authentication: Secure Authentication

## Style Guidelines:

- Primary color: A calming blue (#3498db) for a sense of trust and stability.
- Secondary color: A light gray (#f2f2f2) for backgrounds and subtle accents.
- Accent: A vibrant green (#2ecc71) for positive actions (e.g., income) and highlights.
- Responsive layout adapting to desktop, tablet, and mobile devices.
- Subtle transitions and animations for a smooth user experience.
- Use a consistent and clean icon set (e.g., Material Icons) for clear visual communication.
